# Student Management System (TelebeIdareSistemi)

## Haqqında
Bu layihə Java dilində yazılmış sadə bir tələbə idarəetmə sistemidir. Tələbə və müəllim kimi istifadəçi növləri üçün obyekt yönlü proqramlaşdırma (OOP) prinsiplərindən istifadə edilmişdir.

## Funksionallıqlar
- Tələbə əlavə etmək
- Tələbəni silmək
- Bütün tələbələri göstərmək

## İstifadə olunan siniflər
- `istifadeci`: Əsas istifadəçi sinifi
- `telebe`: `istifadeci`-dən miras alan tələbə sinifi
- `muellim`: `istifadeci`-dən miras alan müəllim sinifi
- `telebeidaresistemi`: Tələbə idarəetməsini həyata keçirən sistem

## Necə işlədir
`main` metodu daxilində tələbələr əlavə olunur, göstərilir və silinir.

## Tələblər
- Java 8 və ya yuxarı
